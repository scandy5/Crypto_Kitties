# Crypto Kitties

## This is a online store of kitties

Demo you can see, if you click [here].

### Technologies that were used in the project:
+ HTML
+ SASS -> CSS
+ JS
+ Gulp

## How to start project
 - open console 
 - `git clone https://github.com/scandy5/Crypto_Kitties`
 - `npm i`
 - `gulp`
